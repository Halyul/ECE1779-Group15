# Assigment 2

This is the source code for ECE1779 2022 Fall Assignment 2

## Flask

### Install Pipenv

```bash
$ pip install --user pipenv
```

### Install Environment

```bash
$ pipenv install
```

### Run

```bash
$ ../start.sh
```

## Frontend

### Install Dependencies

```bash
$ pnpm install
```

### Run Development Server

```bash
$ pnpm run dev
```

### Generate Static Files

```bash
$ pnpm run build
```
